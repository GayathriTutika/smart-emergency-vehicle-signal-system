Prototype Scope

  1. Phone or GPS tracker sends emergency vehicle location every 5 seconds.
  2. Cloud server receives and processes the GPS update.
  3. Server finds the nearest traffic signal from the stored signal list.
  4. If distance is within 1 km, a priority command is generated.
  5. Dashboard shows which signal should change.
  6. Controller box, relay, and real signal switching are future hardware scope.
